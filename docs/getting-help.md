# Getting Help

## GitHub Issues

For bug reports and feature requests, please use [GitHub Issues](https://github.com/vstorm-co/pydantic-ai-backend/issues).

When reporting a bug, please include:

1. Python version (`python --version`)
2. pydantic-ai-backend version (`pip show pydantic-ai-backend`)
3. Minimal code to reproduce the issue
4. Full error traceback

## Discussions

For questions and general discussion, use [GitHub Discussions](https://github.com/vstorm-co/pydantic-ai-backend/discussions).
